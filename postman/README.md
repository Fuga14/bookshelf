# Bookshelf API v1 - Postman Collection

## Опис

Ця колекція містить запити до Bookshelf API v1 для роботи з книгами.

## Встановлення

1. Відкрийте Postman
2. Натисніть "Import" в лівому верхньому куті
3. Виберіть файл `Bookshelf_API.postman_collection.json`
4. Колекція буде імпортована

## Налаштування змінних

Після імпорту колекції, налаштуйте змінну `base_url`:
- За замовчуванням: `http://localhost:3000`
- Для production: встановіть відповідний URL

## Ендпоїнти

### GET /api/v1/books

Отримує список всіх книг з підтримкою пагінації, пошуку та фільтрації.

**Параметри запиту:**
- `page` (опціонально) - номер сторінки (за замовчуванням: 1)
- `per_page` (опціонально) - кількість записів на сторінці (за замовчуванням: 10, максимум: 100)
- `title` (опціонально) - пошук за назвою книги (case-insensitive, частковий пошук)
- `author_id` (опціонально) - фільтр за ID автора

**Приклади:**
- `GET /api/v1/books` - отримати всі книги
- `GET /api/v1/books?page=1&per_page=10` - отримати першу сторінку з 10 книгами
- `GET /api/v1/books?title=Ruby` - пошук книг з "Ruby" в назві
- `GET /api/v1/books?author_id=1` - фільтр за автором з ID=1
- `GET /api/v1/books?title=Ruby&author_id=1` - комбінований фільтр

### GET /api/v1/books/:id

Отримує детальну інформацію про конкретну книгу за її ID.

**Параметри шляху:**
- `id` (обов'язково) - ID книги

**Приклади:**
- `GET /api/v1/books/1` - отримати книгу з ID=1

**Помилки:**
- `404 Not Found` - якщо книга не знайдена

## Формат відповіді

### Успішна відповідь (200 OK)

```json
{
  "id": 1,
  "title": "Ruby on Rails Guide",
  "year": 2020,
  "description": "Comprehensive guide to Ruby on Rails",
  "created_at": "2024-01-15T10:30:00.000Z",
  "updated_at": "2024-01-15T10:30:00.000Z",
  "author": {
    "id": 1,
    "name": "John Doe",
    "bio": "Experienced developer",
    "active": true
  }
}
```

### Помилка (404 Not Found)

```json
{
  "error": "Book not found"
}
```

## Приклади в колекції

Колекція містить наступні приклади відповідей:
- Успішне отримання всіх книг
- Пагінація
- Пошук за назвою
- Фільтр за автором
- Комбіновані фільтри
- Порожній результат
- Книга без автора
- Помилка 404
